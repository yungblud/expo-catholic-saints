## 설명 (Description)

> **Cursed By Jesus**는 가톨릭 성인 정보를 쉽고 빠르게 찾아볼 수 있는 레퍼런스 앱입니다.
>
> **주요 기능**
>
> - 274명의 가톨릭 성인 데이터베이스 (한국 103위 순교성인 포함)
> - 한국어·영어·라틴어 이름으로 퍼지 검색
> - 수호 성인 분야별 검색 (직업, 지역, 질병, 상황 등)
> - 즐겨찾기로 나만의 성인 목록 관리
> - 축일, 생몰년, 시성년도, 전기 등 상세 정보 제공
> - 수호 성인 카테고리별 분류 (직업, 지역, 상황, 질병, 대의, 기타)
>
> **개인정보 보호**
>
> - 모든 데이터는 기기에만 저장됩니다
> - 서버 전송, 계정 생성, 개인정보 수집이 전혀 없습니다
>
> 가톨릭 신자, 교리 공부를 하는 분, 또는 성인에 관심 있는 누구나 활용할 수 있습니다.

## 심사 메모 사항

### 1. 앱 이름 관련 주의 (높은 위험)

**"Cursed By Jesus"**라는 이름은 심사에서 **종교적 불쾌감(religious sensitivity)** 사유로 리젝될 가능성이 높음.

- Apple App Review Guidelines **1.1 - Objectionable Content**: 종교를 비하하거나 모욕하는 콘텐츠 금지
- 예수님의 저주를 의미하는 것으로 오해받을 수 있음
- **이름 변경을 강력히 권장** (예: "가톨릭 성인 도감", "Saints Guide", "성인과 함께" 등)

### 2. 콘텐츠 분류

- **카테고리**: Reference (참고자료) 또는 Education (교육)
- **연령 등급**: 4+ (모든 연령 적합, 부적절한 콘텐츠 없음)

### 3. 개인정보 처리

- 앱 개인정보 보호 질문에서 **"데이터 수집 없음"** 선택
- 개인정보 처리방침 URL: 웹 배포 도메인의 `/privacy-policy` 경로
- 수집 데이터: 없음 (100% 로컬 저장)

### 4. 데모 계정

- 로그인/계정 기능 없음 → 데모 계정 불필요

### 5. 심사 거절 위험 요소

| 위험                      | 수준     | 대응                                                   |
| ------------------------- | -------- | ------------------------------------------------------ |
| 앱 이름 (Cursed By Jesus) | **높음** | 이름 변경 권장                                         |
| 오프라인 전용 / 최소 기능 | 중간     | 검색, 즐겨찾기 등 충분한 인터랙션 존재로 방어 가능     |
| 종교 콘텐츠 정확성        | 낮음     | 공신력 있는 출처 기반, 교리 해석 없이 사실 정보만 제공 |

**권장 스크린샷 구성** (5장):
